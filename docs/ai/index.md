## 基础概念

- Hugging Face 就像 GitHub。
- Model Hub 就像 GitHub 上的代码仓库，只不过存放的是模型。
- Dataset Hub 就像存放数据文件的仓库。
- Transformers 库 就像是一个强大的通用编译器/工具链，让你能轻松运行这些“代码”（模型）。
- Spaces 就像 GitHub Pages，让你能快速展示你的“项目”（AI 应用）。
